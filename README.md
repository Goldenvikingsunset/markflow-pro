## Frequently Asked Questions

**Does it work with split editor groups?**  
Yes. The preview updates whenever you click any `.md` file, regardless of which editor group it's in.

**Does it conflict with GitHub Copilot?**  
No. MarkFlow Pro doesn't register Tab, Ctrl+Enter, or any Copilot keybindings.

**Can I export my Markdown to PDF?**  
Yes — with a Pro licence. Use the Command Palette (`MarkFlow Pro: Export to PDF`) or right-click any `.md` file in the Explorer.

**Can I export to HTML?**  
Yes — with a Pro licence. Exports a self-contained `.html` file with CSS inlined, ready to share.

**Does it support custom themes?**  
Yes — Pro includes GitHub Light, GitHub Dark, Notion-style, and Bear themes.

**Will the preview steal focus from my editor?**  
No. The preview opens without stealing focus so you can keep typing immediately.

**Does it work with frontmatter / YAML metadata?**  
Yes. Frontmatter is rendered visually in the preview panel.
