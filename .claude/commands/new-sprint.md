# Command: New Sprint

## Usage
Run this command at the start of each new sprint.

## Steps
1. Read the current sprint file from sprints/sprint-0X-*.md
2. Read CLAUDE.md to confirm project context
3. Read .claude/rules/extension-architecture.md
4. Read .claude/skills/vscode-extension.md
5. Confirm sprint scope with user before writing any code
6. Implement all deliverables listed in the sprint
7. Run tests mentally / confirm test coverage
8. Mark sprint as COMPLETE in the sprint file
9. Report what was built and what needs manual testing
